**권장 길이**: 2-3분  
**화면 비율**: 16:9 (YouTube 최적화)  
**BGM**: 밝고 경쾌한 게임 음악 (저작권 없는 음악)

## 🎭 시연 시나리오 (3분 버전)

### 📍 오프닝 (0:00 - 0:20)

**화면**: 랜딩 페이지 (https://ggumjisum.vercel.app/)

**내레이션**:
> "여러분, 이번 달도 카드값 폭탄 맞으셨나요?  
> '이번 달은 절약해야지'라고 다짐했지만, 결국 돈은 어디로 갔는지 모르죠.  
> 
> **거지섬**입니다.  
> 지출할수록 섬이 물에 잠기는, 새로운 방식의 예산 관리 앱입니다."

**화면 액션**:
- 랜딩 페이지 Hero 섹션 노출
- 부드럽게 스크롤하며 주요 섹션 훑기
- "앱 시작하기" 버튼에 마우스 호버

---

### 📍 문제 제기 (0:20 - 0:40)

**화면**: 랜딩 페이지 Problem 섹션

**내레이션**:
> "기존 가계부 앱들의 문제, 아시죠?  
> 
> **첫째**, 월 단위는 감각이 없습니다. 30일 뒤는 너무 멀어요.  
> **둘째**, 숫자만 보여주면 지루합니다. 딱딱한 표와 그래프로는 NO.  
> **셋째**, 위기감이 없어서 통제 실패. 숫자만으로는 행동을 바꾸기 어렵죠."

**화면 액션**:
- 3가지 문제점 카드를 하나씩 클로즈업
- 각 카드에 약간의 애니메이션 효과

---

### 📍 온보딩 (0:40 - 1:10)

**화면**: 온보딩 페이지 시작

**내레이션**:
> "자, 그럼 시작해볼까요?  
> 간단한 3단계 설정만 하면 됩니다."

**화면 액션**:
1. **Step 1**: 닉네임 입력
   - 입력: "여행가는거지"
   - "다음" 버튼 클릭

2. **Step 2**: 월 예산 입력
   - 입력: "300,000"
   - 내레이션: "월 30만원 예산이면, 하루에 1만원씩 쓸 수 있겠네요."
   - "다음" 버튼 클릭

3. **Step 3**: 리셋 날짜 입력
   - 입력: "25" (월급날)
   - "시작하기 🏝️" 버튼 클릭

**전환 효과**: 부드러운 페이드 인 → 메인 앱

### 📍 메인 기능 시연 1: 기본 상태 (1:10 - 1:30)

**화면**: 메인 앱 (/island) - 초기 상태

**내레이션**:

> 위에 HP 바처럼 보이는 게 남은 예산입니다.

**화면 액션**:
- 섬 전체 보여주기

- 이웃 섬들 클릭해보기 (1-2개)

### 📍 메인 기능 시연 2: 지출 입력 (1:30 - 2:00)

**화면**: 채팅 입력 시작

**내레이션**:
> "자, 오늘 지출을 입력해볼게요.  
> '스타벅스 5000원'이라고 자연어로 입력하면..."

**화면 액션**:
1. **첫 번째 지출**: "스타벅스 5000원" 입력
   - 엔터 또는 전송 버튼 클릭
   - 거래 내역 표시
   - 물 높이 약간 상승 (5%)
   - 내레이션: "물이 조금 차올랐네요."

2. **두 번째 지출**: "점심 8000원" 입력
   - 물 높이 더 상승 (추가 8%)
   - 내레이션: "점심까지 먹으니 물이 더 차올랐어요."

3. **연속 지출**: "택시 15000원", "저녁 12000원"
   - 빠르게 입력
   - 물 높이 급상승
   - 내레이션: "오늘 좀 썼네요..."

**현재 상태**: 약 40% 사용 (4만원 / 1만원 예산)

---

### 📍 메인 기능 시연 3: 경고 시스템 (2:00 - 2:30)

**화면**: 70% 진입

**내레이션**:
> "자, 여기서 한 번 더 써볼게요.  
> '쇼핑 35000원'!"

**화면 액션**:
1. **70% 돌파**:
   - 입력 즉시 큰 경고 메시지 표시
   - "⚠️ 3시간 24분 동안 2,500원만 써야 함!"
   - 내레이션: "와! 경고가 떴어요!"

2. **5중 시스템 작동**:
   - **날씨**: 맑음 → 폭풍 ⛈️
   - **파도**: 출렁임 3배 빠르게
   - **경고등**: 왼쪽 상단에 삼각형 아이콘
   - **HP 바**: 초록 → 빨강
   - **캐릭터**: 불안해하는 표정
   
   - 내레이션: "날씨가 폭풍으로 바뀌고, 파도가 격렬해지고, HP 바가 빨개졌어요!"

**클로즈업**: 각 요소를 2초씩 확대하여 보여주기

---

### 📍 메인 기능 시연 4: 100% 초과 (2:30 - 2:50)

**화면**: 예산 초과

**내레이션**:
> "그럼... 한 번 더 써볼까요?

   - 큰 오버레이: "예산을 초과했어요 😢"
   - 내레이션: "앗! 캐릭터가 물에 쓸려갔어요!"

2. **복구 퀴즈**:
   - 퀴즈 모달 자동 등장
   - "1000 + 2000 = ?" 같은 간단한 문제
   - 정답 입력
   - 내레이션: "퀴즈를 풀면 70%로 복구할 수 있어요."

3. **복구 완료**:
   - 섬이 다시 떠오름 (70% 지점)
   - 캐릭터 다시 등장
   - 내레이션: "다시 살아났네요!"

## 🎥 촬영 팁

### 화면 녹화 설정
- **해상도**: 1920x1080 (Full HD)
- **프레임**: 60fps (부드러운 애니메이션)
- **브라우저**: Chrome (개발자 도구 열지 않기)
- **마우스 커서**: 표시 (사용자 액션 따라가기)

### 음성 녹음 팁
1. **톤**: 밝고 친근하게, 친구에게 설명하듯이
2. **속도**: 천천히, 명확하게 (1.2배속 재생 고려)
3. **강조**: 핵심 단어는 강하게 (Visual Loss, 일일 예산, 5중 시스템)
4. **감탄사**: 자연스럽게 ("와!", "앗!", "오!")

### 편집 팁
1. **전환**: 부드러운 페이드 인/아웃
2. **강조**: 핵심 기능은 클로즈업 + 줌 인
3. **자막**: 주요 숫자와 키워드만 표시
4. **BGM**: 볼륨 -20dB (내레이션 방해 안 되게)
